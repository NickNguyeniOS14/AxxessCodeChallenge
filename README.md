# AxxessCodeChallenge
iOS Code Challenge for Axxess

# Third party libraries used:
  * Nuke: Download and caching image from the Web
  * Snapkit: Constraint views
  * Realm: Local database storage
  * Alamofire: Make networking calls
  
# Requirements:
  * iOS 13 + 
  * Xcode 12
